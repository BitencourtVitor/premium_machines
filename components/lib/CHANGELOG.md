# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-01-10

### Added
- Initial release of the UI Library.
- **BaseComponent**: Abstract base class with error boundary support.
- **ListingComponent**: Standardized list view with sorting, pagination, and actions.
- **MetricCardComponent**: KPI display card with variants and trend indicators.
- **DesignSystem**:
  - `ThemeProvider` with light/dark mode support.
  - `tokens` for centralized design values.
  - `Icon` component wrapping `react-icons`.
- **Documentation**: README with usage examples.
- **Testing**: Initial test suite for components.
